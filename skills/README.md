# SwiftHerb Skills Directory

This folder contains specialized "skills" or consultants that can help you with various aspects of product development, strategy, and decision-making.

## What are Skills?

Skills are structured knowledge bases and prompt templates that help you:
- Get expert advice on specific topics
- Research solutions and best practices
- Make informed product decisions
- Consult with "virtual experts" on demand

## Available Skills

### 1. Creative Product Manager (`creative-product-manager.md`)
A comprehensive product management consultant that provides:
- Feature suggestions and prioritization
- User experience improvements
- Competitive analysis frameworks
- Product strategy guidance
- Web research capabilities for finding solutions

### 2. Cloudflare Architect Expert (`cloudflare-architect.md`)
A Cloudflare infrastructure specialist that provides:
- Data architecture design for edge computing
- Cloudflare Workers, KV, R2, and D1 optimization
- Data synchronization and update strategies
- Performance optimization for API integrations
- Cost-effective scaling solutions
- Best practices for iHerb product data storage and updates

## How to Use

1. **Consult a Skill**: Open the skill file and use it as a reference or prompt template
2. **Ask Questions**: Use the skill's framework to ask specific questions about your product
3. **Research**: Use the web research capabilities to find real-world examples and solutions
4. **Iterate**: Use the suggestions to improve your product iteratively

## Adding New Skills

To add a new skill:
1. Create a new markdown file in this folder
2. Follow the structure of existing skills
3. Include:
   - Role definition
   - Core competencies
   - Prompt templates
   - Research frameworks
   - Example questions

## Integration with AI

These skills can be used with AI assistants (like ChatGPT, Claude, or DeepSeek) by:
1. Copying the skill's prompt template
2. Adding your specific question or context
3. Using web search for real-world examples
4. Iterating on the suggestions
